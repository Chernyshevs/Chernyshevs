Веб-разработчик
---------------

Мой путь в IT начался с федерального проекта «Код Будущего», где я освоил основы Python. Затем углубил знания на курсах по Python, Django и базам данных. Самостоятельно научился создавать REST API на Flask. Фронтенд изучал под руководством senior-разработчика — теперь работаю с React, TypeScript и Redux Toolkit, пишу тесты и веду документацию в Storybook. Сейчас активно углубляюсь в алгоритмы и олимпиадное программирование в рамках программы от VK Education. Открыт к новым вызовам и интересным проектам!
